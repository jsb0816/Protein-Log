# 💪 Protein Log (프로틴 로그)

> **"쇠질은 안 하고 스마트폰만 보고 있는 거냐? 얼른 몸 움직여라!"**  
> 훈수충 AI 관장님과 함께하는 과학적인 헬스 트레이닝 및 식단 관리 iOS 특화 반응형 웹앱.

Protein Log는 혼자 운동하는 사용자가 가상의 츤데레 전문 트레이너("훈수충")의 실시간 피드백을 받으며, 식단과 운동을 생리학적 공식에 맞춰 체계적으로 관리할 수 있도록 돕는 독립형 웹 애플리케이션입니다. 

별도의 서버 없이 **브라우저 로컬 스토리지(LocalStorage)**를 활용하여 개인의 민감한 신체 정보를 안전하게 보관하며, 사용자가 직접 설정한 **Gemini** 또는 **OpenAI API Key**를 통해 브라우저에서 직접 AI 연동 기능(유튜브 영상 기반 루틴 추출, 냉장고 재료 맞춤형 식단 레시피 생성 등)을 독립 구동합니다.

## 1. 핵심 기능 및 생리학적 연산 공식

### 📊 4대 체형 관리 모드 및 칼로리/단백질 목표 처방
사용자의 목적에 맞는 4가지 모드를 제공하며, 체중 대비 단백질 섭취 비율 및 일일 목표 칼로리를 자동 설정합니다.

| 모드 | 목적 | 칼로리 처방 | 체중 대비 단백질 비율 | 운동 강도 및 유산소 가이드 |
| :--- | :--- | :--- | :--- | :--- |
| **유지어터** | 현재 상태 유지 | 일일 총에너지소비량(TDEE) | 체중 $\times$ 1.2g ~ 1.6g | 주 3-4회 근력 운동, 건강 유지 수준의 유산소 (주 2회, 20분) |
| **린매스업** | 체지방 억제 + 근성장 | TDEE + 250 kcal | 체중 $\times$ 1.8g ~ 2.2g | 주 4-5회 고강도 근력 운동 (점진적 과부하), 유산소 최소화 (주 1-2회, 15분) |
| **벌크업** | 근육량 및 체중 증가 | TDEE + 500 kcal | 체중 $\times$ 2.0g ~ 2.4g | 주 5회 최대 중량 위주 근력 운동, 유산소 최소화 |
| **컷팅 (다이어트)** | 체지방 감소 + 근육 보존 | TDEE - 500 kcal | 체중 $\times$ 1.6g ~ 2.0g | 주 4-5회 높은 볼륨 근력 운동, 필수 유산소 (매 운동 후 30분 이상 인터벌) |

### 🧬 인바디(InBody) 데이터 연동 및 정밀 공식 보정
골격근량(SMM)과 체지방률(PBF)을 입력하면, 단순 성별/나이 기반 수치에서 더욱 발전한 생리학 보정 공식이 가동됩니다.
1. **기초대사량(BMR) 계산 공식 자동 전환**:
   - **인바디 미등록 시**: **미플린-세인트 지올(Mifflin-St Jeor) 공식** 적용
     - *남성*: $10 \times \text{체중(kg)} + 6.25 \times \text{신장(cm)} - 5 \times \text{연령(세)} + 5$
     - *여성*: $10 \times \text{체중(kg)} + 6.25 \times \text{신장(cm)} - 5 \times \text{연령(세)} - 161$
   - **인바디 등록 시**: **캐치-맥아들(Katch-McArdle) 공식**으로 자동 보정
     - 제지방량(LBM) = $\text{체중(kg)} \times (1 - \text{체지방률(\%)} / 100)$
     - $\text{BMR} = 370 + (21.6 \times \text{제지방량(kg)})$
2. **골격근량 기반 단백질 타겟 미세조정**:
   - 골격근량이 표준 이상(남성 체중 대비 38% 이상, 여성 30% 이상)인 경우 단백질 흡수 효율을 고려하여 권장 단백질 범위의 최댓값(예: 린매스업 시 2.2배)으로 목표치를 자동 고정합니다.
3. **골격근량 부족 시 운동 루틴 보강**:
   - 골격근량이 기준 미달(남성 32% 미만, 여성 25% 미만)인 경우, 잔자극 운동 대신 근육량 확충에 가장 효과적인 3대 프리웨이트 다관절 운동(스쿼트, 데드리프트, 벤치 프레스)을 루틴에 우선 자동 배치합니다.

### 💬 "훈수충" AI 피드백 관장 위젯
- 대시보드 상단에 츤데레 헬스장 관장 캐릭터가 항상 상주합니다.
- 오늘 섭취량(칼로리 및 단백질 도달률), 설정된 목표 모드, 쇠질 수행 완료 여부, 인바디 상태를 종합 분석하여 매 시각 칼날 같은 잔소리 멘트를 출력합니다.
- API Key 입력 시 생성형 AI 비서로 가동되며, 키가 없을 때도 다양한 조건 분기형 로컬 스크립트 훈수가 탑재되어 정밀하게 반응합니다.

### 🍎 이원화 식단 로깅 & 냉장고 재료 매칭
- **정밀 분석 모드**: 아침/점심/저녁 먹은 음식명, 칼로리, 탄단지 매크로를 세밀하게 기입하여 누적치를 추적합니다.
- **간편 모드**: 바쁘거나 귀찮은 날에는 터치 버튼과 슬라이더를 이용해 섭취한 단백질 총량(g)만 직관적으로 입력할 수 있습니다.
- **냉장고 보유 재료 연동**: 집에 있는 재료(닭가슴살, 현미밥, 계란 등)를 등록하고 `AI 보유 재료 식단 추천 생성`을 탭하면, 냉장고 재료를 최우선으로 매칭하여 오늘 영양 가이드에 딱 맞는 단백질 식단 레시피를 AI가 처방해 줍니다.

### 🏋️ 유튜브 링크 기반 루틴 자동 추출
- 유튜브 운동 루틴 영상 링크(URL)와 영상 설명글 또는 자막을 복사해 넣으면, Gemini/OpenAI API를 통해 영상 내 소개된 운동 명칭, 세트수, 반복 횟수, 자세 팁을 JSON 구조 데이터로 분석하여 내 운동 리스트에 1초 만에 자동 이식합니다.

## 2. 모바일 최적화 UI/UX 디자인 시스템

본 애플리케이션은 **White 70% & Sky-Blue 30%**의 세련되고 청량감 있는 감각적인 피트니스 테마로 구축되었습니다.

1. **iOS Native-like UI/UX**:
   - iPhone의 노치(Notch) 영역과 하단 홈 바(Home Bar) 간섭을 원천 차단하는 `safe-area-inset-*` 레이아웃 및 뷰포트 스크롤을 고정하였습니다.
   - iOS 스타일의 Bottom Sheet(하단 슬라이드 업 모달) 컴포넌트와 토글 애니메이션을 제공합니다.
   - 버튼 클릭 시 미세하게 튕기는 **iOS 특화 햅틱 피드백 애니메이션(`ios-btn-press`)**이 내장되어 있습니다.
2. **반응형 기기 대응**:
   - 데스크톱/태블릿 화면으로 접속할 경우 화면 중앙에 **물리 폰 쉘(Notch, 시간 정보, 5G 신호 안테나, 배터리 아이콘, 하단 홈 바가 렌더링된 아이폰 목업 프레임)** 형태로 스마트하게 표현되어 기기에 상관없이 균일한 뷰포트를 제공합니다.
   - 실제 모바일 기기(iPhone 등)로 접속할 때는 목업이 사라지고 전체 모바일 뷰로 깔끔하게 렌더링됩니다.
3. **PWA(Progressive Web App) 대응**:
   - 모바일 브라우저에서 '홈 화면에 추가'를 할 경우 독립 실행형 네이티브 앱처럼 브라우저 주소창 없이 풀 스크린으로 동작합니다.

## 3. 기술 스택 (Tech Stack)

- **프레임워크**: React (Vite, TypeScript)
- **스타일링**: Tailwind CSS v4.0 (청량한 하늘색 포인트 `#0EA5E9` 및 슬레이트 폰트 테마)
- **아이콘 라이브러리**: Lucide React
- **상태 관리**: React Context API
- **데이터 저장**: LocalStorage 브라우저 데이터 동기화
- **인공지능 API**: Gemini (gemini-2.5-flash) 및 OpenAI (gpt-4o-mini) 연동

## 4. 시작 가이드 (Getting Started)

### 사전 요구 사항
컴퓨터에 [Node.js](https://nodejs.org/)가 설치되어 있어야 합니다.

### 설치 및 로컬 서버 실행
1. 저장소 클론 및 폴더 이동:
   ```bash
   git clone https://github.com/jsb0816/Protein-Log.git
   cd Protein-Log
   ```
2. 의존성 패키지 설치:
   ```bash
   npm install
   ```
3. 개발 서버 기동:
   ```bash
   npm run dev
   ```
   실행 후 브라우저에서 `http://localhost:5173`으로 접속할 수 있습니다.

### 프로덕션 빌드 및 정밀 검증
```bash
npm run build
```
빌드 완료 시 `dist/` 폴더에 최적화된 HTML/JS/CSS 결과물이 생성됩니다.
